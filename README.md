# CutForge - Professional Video Editor

A cross-platform video editing suite built with Tauri v2 (Rust + React).

## Vision
- Professional-grade video editing (Premiere Pro/Resolve/FCP level)
- Native cross-platform: Linux, Windows, macOS
- Real-time preview with WebGPU/WebGL
- FFmpeg/GStreamer backend for video processing
- React 18 + TypeScript frontend
